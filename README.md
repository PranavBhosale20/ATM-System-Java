# ATM System (Java + JDBC + MySQL)

A console-based ATM transaction system developed using Java and MySQL.

## Features
• Customer Login  
• Admin Login  
• Deposit Money  
• Withdraw Money  
• Balance Enquiry  
• Transaction History  
• Multiple User Accounts  

## Technologies Used
Java  
JDBC  
MySQL  

## Project Structure
ATM-System-Java
src/atm_project
database
screenshots

## How to Run
1. Import project into Eclipse
2. Configure MySQL connection in DatabaseConnection.java
3. Run ATMSystem.java
